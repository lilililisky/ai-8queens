# 调试记录

## 问题1：数据集下载失败
- **现象**：kaggle CLI 报错 KeyError: 'username'
- **解决**：手动从 Kaggle 网站下载数据集

## 问题2：验证集准确率波动
- **现象**：第4轮验证准确率从97%降到91%
- **原因**：学习率过大导致震荡
- **解决**：加入 ReduceLROnPlateau 调度器

## 问题3：测试集准确率偏低
- **现象**：测试集准确率78%，远低于验证集98%
- **原因**：过拟合，数据量不足
- **解决**：建议增加数据增强、Dropout层

## 最终结果

- 测试准确率：78.85%
