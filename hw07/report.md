## 二、模型结构

使用 **ResNet18** 预训练模型，修改最后全连接层为二分类。

| 层 | 输入 | 输出 | 参数量 |

| Conv1 | 3x224x224 | 64x112x112 | 9,408 |

| **总计** | | | **~11.2M** |

- Epochs: 5

- Optimizer: Adam

**测试准确率：78.85%**

## 五、结果分析

1. **高准确率但召回率偏低**：NORMAL 类的召回率仅44%，说明模型容易把正常图片误判为肺炎。从医学诊断角度，**召回率（不漏诊）更重要**，因为把肺炎误判为正常可能导致患者延误治疗。

2. **过拟合现象**：训练集准确率98%，测试集仅79%，说明模型记住了训练数据但泛化能力不足。建议增加数据增强、使用更强的正则化。

3. **假阴性的后果**：将肺炎误判为正常（假阴性）会延误治疗，后果严重；将正常误判为肺炎（假阳性）会导致不必要的检查和焦虑。

## 六、改进建议

- 使用更多的数据增强
- 尝试更深的网络（ResNet50）
- 使用 GPU 加速训练
- 增加 Epoch 数量
- 使用迁移学习 + 微调
