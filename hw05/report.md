# 实验报告：手写数字识别（CNN vs LeNet-5）

## 一、文章学习摘要

通过学习微信公众号文章《计算机视觉》第10篇，了解了卷积神经网络（CNN）的基本结构：
- **卷积层**：提取图像局部特征
- **池化层**：降低维度，减少参数量
- **全连接层**：分类输出

CNN 通过层层提取特征，能够有效识别手写数字，MNIST 数据集是经典的入门实验。

## 二、极简 CNN 结构

| 层 | 输入尺寸 | 输出尺寸 | 参数量 |
|---|---|---|---|
| Conv1 | 1x28x28 | 32x28x28 | 320 |
| Pool1 | 32x28x28 | 32x14x14 | 0 |
| Conv2 | 32x14x14 | 64x14x14 | 18,496 |
| Pool2 | 64x14x14 | 64x7x7 | 0 |
| FC1 | 3136 | 128 | 401,536 |
| FC2 | 128 | 10 | 1,290 |
| **总计** | - | - | **~42万** |

## 三、LeNet-5 结构

LeNet-5 是 Yann LeCun 1998 年提出的经典网络，用于手写数字识别。

| 层 | 输入尺寸 | 输出尺寸 | 参数量 |
|---|---|---|---|
| Conv1 | 1x28x28 | 6x28x28 | 156 |
| AvgPool1 | 6x28x28 | 6x14x14 | 0 |
| Conv2 | 6x14x14 | 16x10x10 | 2,416 |
| AvgPool2 | 16x10x10 | 16x5x5 | 0 |
| FC1 | 400 | 120 | 48,120 |
| FC2 | 120 | 84 | 10,164 |
| FC3 | 84 | 10 | 850 |
| **总计** | - | - | **~6万** |

## 四、训练结果对比

| 模型 | 参数量 | 测试准确率 | 5轮训练耗时 |
|---|---|---|---|
| SimpleCNN | ~42万 | 99.11% | ~150秒 |
| LeNet-5 | ~6万 | 98.82% | ~80秒 |

超参数设置：
- Epochs: 5
- Batch Size: 64
- Learning Rate: 0.001
- Optimizer: Adam
- Device: CPU

## 五、结论

1. **参数量对比**：LeNet-5（~6万）约为 SimpleCNN（~42万）的 **1/7**
2. **准确率对比**：两者均达到 **98%+**，相差仅 0.29%
3. **训练速度**：LeNet-5 比 SimpleCNN 快约 **45%**

**分析**：
- 极简 CNN 使用更深的通道（32→64），参数量更大，准确率略高
- LeNet-5 设计精炼，用更少的参数达到相近效果，训练更快
- 对于 MNIST 这种简单任务，LeNet-5 已足够优秀
